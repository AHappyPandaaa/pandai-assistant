====================================
 PandAI Assistant - Installation Guide
====================================

REQUIREMENTS
------------
- Windows 10 or 11
- NVIDIA GPU (recommended for fast transcription)
- Internet connection for first-time setup (~1.5 GB download)
- An Anthropic API key (free at console.anthropic.com)

FIRST TIME SETUP
----------------
1. Extract this folder somewhere on your computer (e.g. Desktop or C:\)
2. Double-click "install.bat"
3. Wait for setup to complete (5-10 minutes)
4. A "PandAI Assistant" shortcut will appear on your Desktop

LAUNCHING THE APP
-----------------
- Double-click the "PandAI Assistant" shortcut on your Desktop
- OR double-click "launch.bat" inside this folder

ADDING YOUR API KEY
-------------------
1. Go to console.anthropic.com and sign up for a free account
2. Create an API key
3. Open PandAI Assistant and click the Settings button (⚙)
4. Paste your API key and click Save

NOTES
-----
- The first launch may take 30-60 seconds while the AI model loads
- Audio routing software (e.g. VB-Cable) is needed to capture system audio
- Your API key is stored locally and never shared
